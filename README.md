# picker

Deno app: Hono (HTTP), Zod, ts-morph (AST), Deno KV. Entry: `main.ts`.

## Run

- **Dev**: `deno task dev`
- **Once**: `deno run --allow-net --unstable-kv main.ts`
- **Test**: `deno test`
